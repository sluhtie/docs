Claiming System ist Chunkbasiert Über ein Verwaltungs GUI können Flags und Nutzergruppen gesetzt werden.

**Rollen & Permissions System:** Über das GUI können Spieler Claim Rollen erstellen und die Berechtigungen von dieses ebenfalls bearbeitenDie Anzahl an Rollen hängt von der Permission des Spielers ab.

**Claim Flags**:
\- **Build:** Erlaubt das abbauen und platzieren von Blöcken.
\- **AnvilUse:** Erlaubt das verwenden von ambossen.
\- **Explosions:** Erlaubt das explodieren von TNT.
\- **Farmland-Trample:** Erlaubt das zertrampeln von Beeten.
\- **MobGriefing:** Erlaubt das griefen von Mobs. (Creeper, Enderman)
\- **MobDamage:** Erlaubt das attakieren von Mobs.
\- **MobSpawning:** Erlaubt das natürliche Spawnen von Mobs.
\- **PvP:** Erlaubt das PvP.
\- **ItemPickup:** Erlaubt das aufheben von Items.
\- **ItemDrop:** Erlaubt das droppen von Items.
\- **EnderPearl:** Erlaubt das Teleportieren mit Enderperlen.

\- Willkommens Nachricht beim Chunk Entern (An, Aus)
\- Use Flag

Für das Interagieren mit blöcken sollte es für jegliche blöcke die option geben die Interaction zu aktivieren und deaktivieren**Befehle**

* `/claim info` - Claim Informationen.

* `/claim list` - Liste deiner Claims. (GUI)

* `/claim create` - Erstelle einen neuen Claim.

* `/claim delete` - Lösche deinen Claim.

* `/claim expand` - Erweitere deinen Claim um nebenstehenden Chunk

* `/claim unclaimchunk` - Entclaimt den Chunk auf dem du stehst.

* `/claim add <Spieler>` - Füge einen Spieler zum Claim hinzu

Nur während der Inhaber online ist, kann dieser das claim bebauen etc (So wie in plotsqwuared)

* `/claim trust <Spieler>` - Vertraue einem Spieler.

(Kann dann auch abbauen während der Owner offline ist.)

* `/claim untrust <Spieler>` - Entziehe einem Spieler die rechte von einem Claim.

* `/claim ban <Spieler>` - Banne einen Spieler von seinem claim.

(Jegliche teleportationen in diese Area sollten ebenfalls unterbunden werden. Homes, Tpa etc)

* `/claim rename` \<Name> - Gebe deinem Claim ein namen.

* `/claim transfer` \<Spieler> - Übertrage dein Claim auf einen Spieler.

* `/claim show` - Zeige die Chunk Outlines.

* `/claim manage` - Verwalte deinen Claim.

**Administrator Befehle**

* `/claim debug` - Debug Informationen. (Dev)
